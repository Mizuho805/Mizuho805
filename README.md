## 🏛 Philosophy: Data Sovereignty & Agnosticism

AI ツールは「有能なエンジン」に過ぎません。私は以下の原則に基づき、自身の「母艦（System）」を運用しています。

- **Local First**: すべての思考ログは Markdown としてローカル環境に保管し、所有権を維持。
- **Tool-Agnostic**: 各社の最新 AI を「プラグイン」として使い分け、特定のツールにデータを人質に取らせない。
- **ANKP (AI-Native Knowledge Protocol)**: 自然言語で定義された独自のデータ規格。どの AI でも即座に私の文脈を理解し、高度な推論を可能にする「ポータブルな知能」を実現。

## 🚀 Framework: Personal Intelligence System

AI との共進化（Co-evolution）を目的とした個人用オペレーティング・フレームワーク。

- **Issue-Driven Evolution**:
  - すべての改善プロセスを GitHub Issue として管理。AI エージェントと共に「1 セッション・1 コミット」のサイクルを回し、日々の進化を緑の履歴（芝生）として刻んでいます。
- **Evolution Loop**:
  - 修正指示を単なる「直し」で終わらせず、プロトコルや仕組み（Skill）へと還元し続ける自己進化型ワークフロー。
- **Structural Resilience**:
  - ツールが変わっても「思考の構造」は変わらない。Markdown による構造化能力そのものが、最大の資産であると定義しています。
